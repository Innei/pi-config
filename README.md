# pi-config

My [pi](https://github.com/mariozechner/pi-coding-agent) configuration.

## Setup

Clone to `~/.pi`:

```bash
git clone git@github.com:Innei/pi-config.git ~/.pi
```

## Settings

| Key | Value |
|---|---|
| Theme | `catppuccin-mocha` |
| Default Model | `mimo-v2.5-pro` |
| Thinking Level | `high` |
| Quiet Startup | `true` |
